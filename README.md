# 🛡️ Mini Intrusion Detection System (IDS Lite)

## 📌 Description
A simple intrusion detection system that simulates network traffic monitoring and detects suspicious activity based on request frequency.

This project demonstrates basic cybersecurity monitoring and threat detection concepts.

## 🎯 Features
- Simulated network traffic analysis
- Detects suspicious IP activity
- Threshold-based alert system
- Real-time style monitoring output

## 🛠️ Technologies Used
- Python
- Collections (defaultdict)




## 🧠 What I Learned
- Intrusion detection basics
- Network traffic monitoring
- Behavior-based threat detection
- Security alert systems
